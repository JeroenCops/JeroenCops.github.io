Cobalt

This website was generated with Cobalt, which is a static website generator similar to Jekyll. Cobalt is written in Rust, so it Cobalt can turn the Liquid templates, MarkDown files almost instantly into HTML. Because it's static makes it also easy to deploy anywhere, you don't have to mess with SQL and php, I just can focus on the content.
