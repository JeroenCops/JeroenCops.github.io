The Rust book

Technologie used mdbook

Rust is a systems programming language, that is focused on safety, speed, and concurrency. It achieves all these goals without having a garbage collector. It provides a number of compile-time safety checks that produce no runtime overhead, while eliminating all data races. This is the Dutch translation of the book. So have fun!
